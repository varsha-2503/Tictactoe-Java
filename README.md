# Tic Tac Toe (Java)

This is a simple console-based Tic Tac Toe game written in Java.  
It is a beginner-friendly project created to practice basic Java programming concepts.

The game allows two players (X and O) to take turns, validates user input, displays the board after each move, and checks for win or draw conditions.

## How to Run
Make sure Java (JDK) is installed on your system, then run:
javac tictactoe.java
java tictactoe

## Concepts Used
- Arrays
- Loops and conditional statements
- Methods
- Exception handling
- Ternary operator
